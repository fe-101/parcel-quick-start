## 使用
下载安装
```
$ git clone git@github.com:haloislet/parcel-quick-start.git 
$ cd parcel-quick-start
$ npm install
```

启动开发环境
```
$ npm run dev 
```
构建生产环境
```
$ npm run build
```
启动本地服务器查看生产环境
```
$ npm run start
```
