> vue.js 项目模板,使用了 parcel 作为打包工具
<br>

## 使用
下载安装
```
$ git clone -b vue git@github.com:haloislet/parcel-quick-start.git
$ cd parcel-quick-start
$ npm install
```

启动开发环境
```
$ npm run dev 
```
构建生产环境
```
$ npm run build
```
启动本地服务器查看生产环境
```
$ npm run start
```

<br>

## 说明
使用了以下包
- vue
- vuex
- vue-router
- axios
- postcss
- babel
- http-server
